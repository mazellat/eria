LECTEUR DE CARTE ERIA
MAZELLA Thomas & SICSIC Robin

Utilisation:
1- Brancher le lecteur à un ordinateur
2- Lancer l'application et se connecter au lecteur via le sélécteur de port série et le bouton "Connexion"

Paiement :
3- Entrer un montant sur le clavier du boîtier pour saisir un montant à débiter d'un compte
4- Passer le badge correspondant sur le lecteur de badge RFID

Consultation de solde:
3- Passer un badge sur le lecteur

Ajouter un utilisateur:
3- Cliquer sur "Ajouter un utilisateur" sur l'application windows
4- Entrer les informations relatives au nouvel utilisateur
5- Valider et passer un badge non enregistré sur le lecteur RFID du boîtier
